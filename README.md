# 项目人员调度台

一套可在 Windows 10/11 离线使用的项目资料库、人员资料库与产能调度桌面软件。

## 已实现功能

- 项目资料库：项目基础信息、客户、DDL、剧本与美术资料、资产/视频/总进度、内审、风险、本地路径等。
- 人员资料库：部门与职位、技能等级、分方向制作能力、AI/其它部门项目、逐项目产能占用、释放日期与在岗状态。
- 双向关联：项目团队与人员参与项目均来自同一组“分工记录”，不会出现两边数据不一致。
- 阶段状态：支持资产制作中/完成、视频制作中/完成、反馈修改中等状态，并在阶段缺少对应人员时醒目提醒。
- 核心团队：项目负责人/导演、PM、美术监制、视频制作人员、资产制作人员五个岗位均至少一人，也支持多人协作。
- 阶段产能释放：资产制作完成后自动释放资产人员；导演和视频制作人员仅在项目完成/取消或移出项目后释放。
- 调度面板：建立项目用人需求，按职能和剩余产能推荐人员，分配时提示是否超额。
- 风险总览：展示项目进度、DDL、风险状态、人员负载和未满足用人需求。
- 进度优先界面：工作台集中展示总进度、资产进度、视频进度；项目列表将主要空间用于五岗位团队与人员调度，项目基础资料折叠收纳。
- Excel / CSV 批量导入：提供项目和人员 `.xlsx` 模板；支持重名更新和缺失人员自动建档。
- 离线备份：完整数据库导出/恢复为 JSON 文件。
- 单机本地数据：数据库保存在当前 Windows 用户的应用数据目录，不联网、不上传。

## Windows 使用

构建结果包含：

- `项目人员调度台-Setup-1.3.0-x64.exe`：安装版，可选择安装目录并创建桌面快捷方式。
- `项目人员调度台-Portable-1.3.0-x64.exe`：便携版，无需安装，双击运行。
- `项目资料导入模板.xlsx`、`人员资料导入模板.xlsx`。

## 本地开发

```bash
npm install
npm run templates
npm start
```

测试：

```bash
npm test
```

## 生成 Windows 安装包

在 Windows 电脑执行：

```bash
npm ci
npm run templates
npm run build:win
```

也可以在 GitHub 仓库的 `Actions` 页面手动运行 `Build Windows Installer`，完成后在该次运行底部下载 Windows 构建产物。

## 数据安全

首次正式录入前可先使用少量测试记录熟悉操作，确认后清空测试记录再开始。正式使用后建议每周导出一次完整备份。
