# 项目人员调度台

一套可在 Windows 10/11 离线使用的项目资料库、人员资料库与产能调度桌面软件。

## 已实现功能

- 项目资料库：项目基础信息、客户、DDL、剧本与美术资料、资产/视频/总进度、内审、风险、本地路径等。
- 人员资料库：部门与职位、技能等级、分方向制作能力、AI/其它部门项目、逐项目产能占用、预计释放日期与在岗状态；预计日期仅供排期参考，可调度状态以全部有效项目的剩余产能为准。
- 多职位与筛选：每人可选择多个职位；人员总数、可调度、超负荷和非在岗统计卡可直接筛选对应人员。
- 双向关联：项目团队与人员参与项目均来自同一组“分工记录”，不会出现两边数据不一致。
- 阶段状态：支持资产制作中/完成、视频制作中/完成、反馈修改中等状态，并在阶段缺少对应人员时醒目提醒。
- 核心团队：项目负责人/导演、PM、美术监制、视频制作人员、资产制作人员五个岗位均至少一人，也支持多人协作。
- 阶段产能释放：资产制作完成后自动释放资产人员；导演和视频制作人员仅在项目完成/取消或移出项目后释放。
- 调度面板：缺员按项目卡汇总展示；候选列表包含全部人员，满载/超载人员仍可经风险确认后分配。
- 离线账号与权限：人员姓名作为登录账号，首次使用随机临时密码；总经理和 PM 可调度并维护业务资料，其余人员只读；高级管理员 `admin` 可维护账号、选项字典与自定义资料字段。
- 风险总览：展示项目进度、DDL、风险状态、人员负载和未满足用人需求。
- 团队优先界面：项目列表压缩进度区域、加大关键信息字号，优先级使用不换行角标，长项目名自动省略；默认按进行中、紧急程度、接单日期、待启动及已完结顺序排列。
- Excel / CSV 批量导入：提供项目和人员 `.xlsx` 模板；支持重名更新和缺失人员自动建档。
- 离线备份：完整数据库导出/恢复为 JSON 文件。
- 单机本地数据：数据库保存在当前 Windows 用户的应用数据目录，不联网、不上传。

## Windows 使用

构建结果包含：

- `项目人员调度台-Setup-1.5.0-x64.exe`：安装版，可选择安装目录并创建桌面快捷方式。
- `项目人员调度台-Portable-1.5.0-x64.exe`：便携版，无需安装，双击运行。
- `项目资料导入模板.xlsx`、`人员资料导入模板.xlsx`。

## 本地开发

```bash
npm install
npm run templates
npm start
```

测试：

```bash
npm test
```

## 生成 Windows 安装包

在 Windows 电脑执行：

```bash
npm ci
npm run templates
npm run build:win
```

也可以在 GitHub 仓库的 `Actions` 页面手动运行 `Build Windows Installer`，完成后在该次运行底部下载 Windows 构建产物。

## 数据安全

首次启动时，登录页会显示高级管理员 `admin` 的一次性随机临时密码；登录后必须立即修改。人员建档后也会生成各自的随机临时密码，高级管理员可在账号管理中查看或重置。正式密码经加盐加密后保存在当前电脑，不联网、不上传。

首次正式录入前可先使用少量测试记录熟悉操作，确认后清空测试记录再开始。正式使用后建议每周导出一次完整备份。
